Calculadora Básica.

Descrição:
Calculadora básica feita em java, com o objetivo de treinar a logica de programação, onde ela realiza as principais operações matemáticas e possui uma interface gráfica.

Funcionalidades:
-soma.
-subtração.
-divisão.
-multiplicação.
-Potenciação.
-raiz quadrada.



Autor: Marcus Vinicius Mendes.
